# devops-cloud-management
The devops-webapp project is a simple static web application developed to demonstrate end-to-end DevOps automation using modern CI/CD practices. The application is version-controlled using GitHub, where every code change automatically triggers a Continuous Integration (CI) pipeline powered by GitHub Actions.
